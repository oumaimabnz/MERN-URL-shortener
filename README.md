# 🔗 MERN URL Shortener

A **full-stack URL shortener application** built using the **MERN stack** (MongoDB, Express, React, Node.js).  
This app allows users to generate short URLs from long links, redirect to the original URLs efficiently, generate a QR code for each provided link, and download the QR code image.

The project demonstrates backend API development, database integration, frontend-backend communication, and user-focused features in a real-world use case.

## 🧰 Tech Stack

### Frontend
- **React**
- **JavaScript (ES6+)**
- **TailwindCSS**

### Backend
- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**

---

## ✨ Features

- 🔗 Shorten long URLs into unique short links  
- 🚀 Redirect users from short URLs to original URLs  
- 📱 Generate QR codes for shortened URLs  
- ⬇️ Download QR code images  
- 🗂 Store URLs in MongoDB  
- ⚡ RESTful API for URL creation and redirection  
- 📱 Simple and responsive user interface
